# Ignews

Ignews é uma página estática de um blog JAMstack em Nextjs que utiliza API exterma e autenticação com o github.

### Current Stack :technologist:
- `Nextjs`
- `Next-Auth`
- `Prismic`
- `FaunaDB`
- `Stripe`
